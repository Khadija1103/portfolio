# Portafolio Profesional

## Descripción

Este es un portafolio web personal desarrollado con HTML, CSS y JavaScript, donde se muestran proyectos, habilidades y experiencia en desarrollo web y análisis de datos.

El objetivo del portafolio es presentar de manera clara y organizada los trabajos realizados, demostrando conocimientos en diseño web, lógica de programación y manejo de tecnologías frontend.

## Secciones del portafolio

* Página principal con presentación personal.
* Sección “Acerca de mí”.
* Proyectos desarrollados.
* Habilidades técnicas.
* Información de contacto.
* Diseño responsive adaptable a diferentes dispositivos.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
